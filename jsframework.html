Brief

To apply knowledge of React to build an eCom store.

Requirements

  • Homepage: List all products with a searchable, filterable list.
  • Individual Product Page: Display product details, reviews, and add to cart functionality.
  • Cart Page: List all cart items and display total.
  • Checkout Success Page: Confirm order success and clear the cart.
  • Contact Page: Include a validated contact form.
  • Header: Navigation bar and Cart icon displaying item count.
  • Product Card: Displays on Homepage with a "View Product" button.
  • Homepage: Search bar for product filtering; clicking a product leads to its detailed page.
  • Product Page: Shows product details, discounted price, and allows adding to cart.
  • Cart Page: Lists cart items and total; includes a checkout button.
  • Checkout Success Page: Displays order success message and clears the cart.
  • Contact Page: Form with validation for full name, subject, email, and message body.

Technical

  • Noroff API
  • Frontend Framework: React
  • Routing: React Router for navigation between pages
  • State Management: React useState and useEffect hooks
  • Styling: CSS
  • Hosted on Netlify
